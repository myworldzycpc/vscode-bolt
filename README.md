# bolt README

This is a Visual Studio Code extension that provides a set of tools for managing Bolt projects.

> The development of this extension is currently underway and there are some incomplete features.

## Features

- Semantic highlighting for Bolt files.

## Requirements

- Python (Developed and tested with Python 3.14, but should work with some earlier versions as well)
- `beet` `bolt` `mecha` installed (can be installed using `pip install beet bolt mecha`)

## Extension Settings

Not available at the moment.
